# Code of Conduct

## The short version

Be respectful. We're all here to build something useful. Harassment, trolling, and personal attacks aren't tolerated.

## The longer version

We want tempograph's community to be welcoming to everyone, regardless of experience level, background, identity, or any other personal characteristic.

**Do:**
- Be kind and constructive in feedback
- Assume good intent
- Help people learn when they ask questions
- Respect differing opinions and approaches

**Don't:**
- Use sexualized language or imagery
- Troll, insult, or make personal attacks
- Harass anyone, publicly or privately
- Share others' private information without permission

## Enforcement

If someone is making the community worse, report it to eali@needspec.com. Reports will be reviewed and acted on. Maintainers who don't follow the Code of Conduct may be removed from the project.

## Scope

This applies everywhere the project lives: GitHub issues, PRs, discussions, and any other communication channels.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
